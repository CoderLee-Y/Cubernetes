## Quick start

Requirements: 

- Etcd(suggested path: /usr/local/bin)
- Nginx
- Resolvconf
- Kafka

Enjoy Cubernetes!

```shell
./build/cuberoot init -f ./example/yaml/master-node.yaml
./build/cuberoot join $master_ip -f ./example/yaml/slave-node.yaml
```
